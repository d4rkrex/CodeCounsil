# UX Checklist

- Accessibility coverage against WCAG 2.1 AA (labels, semantics, focus order, announcements)
- Keyboard navigation, focus visibility, and escape/submit behaviors
- Color contrast and non-color-only status communication
- Form usability: labels, defaults, validation timing, and helpful error text
- Responsive layouts across mobile, tablet, and desktop breakpoints
- Loading, empty, success, and error states for user-facing views
- User-flow friction, dead ends, and confusing navigation or hierarchy
